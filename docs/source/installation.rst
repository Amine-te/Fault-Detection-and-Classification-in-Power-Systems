Installation
============

Requirements
------------
- Python 3.8+
- TensorFlow 2.10+
- scikit-learn 1.2+
- Streamlit 1.22+

Install from source:

.. code-block:: bash

   git clone https://github.com/yourusername/power-system-anomaly-detection.git
   cd power-system-anomaly-detection
   pip install -r requirements.txt
